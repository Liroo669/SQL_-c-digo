Estudos de SQL Server

Este repositório foi criado com o objetivo de armazenar exercícios e consultas SQL desenvolvidos durante meus estudos de Banco de Dados.

Atualmente, o repositório contém exemplos práticos utilizando SQL Server, abordando conceitos fundamentais para análise e manipulação de dados.

Conteúdos estudados

- Comando WHERE
- Operadores AND e OR
- Filtragem de dados
- Função SUM()
- Consultas SELECT

Tecnologias utilizadas

- SQL Server

## Objetivo

Consolidar meus conhecimentos em Banco de Dados e SQL por meio da prática, registrando minha evolução durante a graduação em Análise e Desenvolvimento de Sistemas.
